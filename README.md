## Build and Execute Instructions:
To prepare this program for execution, first download Xampp. After downloading Xampp, a folder named "xampp" should be appear on your Local Disk. In the "xampp" folder, there is a folder named "htdocs". Git clone this repository into that "dtdocs" folder. Now, go to the Xampp control panel and start your MySQL server. Then, go to the shell in the Xampp control panel and type mysql -u root. Copy and paste all the SQL code in projectmanager.sql file into your Xampp shell and then click enter. Now, in the Xammp control panel, start Apache. The program is now on localhost. To see it, type localhost/ProjectManager in your browser's search bar.
